# FAIRmodels-backend

## Getting Started

### 1. Create and activate virtual environment

```
python3 -m venv .venv
source .venv/bin/activate
```

### 2. Execute migrations

```
python manage.py migrate
```

### 3. Create admin account

```
python manage.py createsuperuser
```

### 4. Run server

```
python manage.py runserver
```
